## Realiza el ejercicio

Buscamos un 'Hello World' funcional. El backend debe levantar en un puerto, el frontend debe mostrar una pantalla básica y la base de datos debe estar configurada en un contenedor local o en la nube, pero sin tablas ni datos todavía. Puedes elegir las tecnologías que quieras. El proyecto debe poder arrancarse sin problemas, tanto en frontend como en backend. No hace falta crear ninguna funcionalidad.

Dado que tienes que construirlo de 0, no hay semilla del proyecto. Es más una invitación a experimentar en esta primera fase del proyecto con tecnologías que uses en tu día a día, de manera que estés preparado para comenzar un proyecto nuevo en el futuro.
