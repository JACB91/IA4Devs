En la sesión en vivo de este módulo haremos diferentes ejercicios en bases de datos usando asistentes IA. Para estar preparados para la sesión y usar de manera eficaz nuestro tiempo, te proponemos tener la herramienta descargada, conectar con una base de datos remota, habituarte a su interfaz, y tratar de lanzar alguna consulta.
